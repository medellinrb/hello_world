# HelloWorld

Print out the classic "hello world!" message

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hello_world'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hello_world

## Usage


```ruby
HelloWorld.hi
"Hello mundo!"
```

```ruby
HelloWorld.person('John Doe')
"Hello John Doe"
```
Simple isn't?
